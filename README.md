# 🛒 Retail Weekly Review Dashboard — Power BI

A high-level, interactive Weekly Review Dashboard built in Power BI for a retail firm as part of the Data Visualization module assignment.

---

## 📊 Dashboard Overview

This dashboard provides management with a complete weekly sales performance review, with the ability to slice data by time intervals and city tiers.

---

## 📁 Files

| File | Description |
|---|---|
| `Retail_Weekly_Dashboard.pbix` | Power BI dashboard file |
| `retail_sales_data.csv` | Sample retail sales dataset (1500 rows) |

---

## 📈 Visuals Included

| Visual | Description |
|---|---|
| KPI Cards | Total Revenue, Units Sold, Units Returned |
| Sales Trend Chart | Revenue over time by City Tier |
| Donut Chart | Revenue split by Product Category |
| Bar Chart | Revenue comparison across City Tiers |
| Clustered Bar Chart | Units Sold vs Units Returned by Month |
| Slicers | Filter by City Tier, Category, Quarter |

---

## 🗂️ Dataset Columns

| Column | Description |
|---|---|
| Date | Transaction date |
| Week_Number | Week number of the year |
| Month | Month name |
| Quarter | Q1, Q2, Q3, Q4 |
| Year | Year |
| City_Tier | Tier 1, Tier 2, Tier 3 |
| City | City name |
| Category | Product category |
| Product_Name | Product name |
| Unit_Price | Price per unit (₹) |
| Units_Sold | Number of units sold |
| Discount_Pct | Discount percentage applied |
| Revenue | Total revenue after discount (₹) |
| Units_Returned | Number of units returned |

---

## 🏙️ City Tiers

| Tier | Cities |
|---|---|
| Tier 1 | Mumbai, Delhi, Bangalore, Chennai, Hyderabad |
| Tier 2 | Pune, Ahmedabad, Jaipur, Lucknow, Surat |
| Tier 3 | Coimbatore, Indore, Nagpur, Bhopal, Patna |

---

## 🛠️ How to Open

1. Download and install **Power BI Desktop** (free) from [microsoft.com/power-bi](https://www.microsoft.com/en-us/power-bi)
2. Download the `Retail_Weekly_Dashboard.pbix` file
3. Open it in Power BI Desktop
4. Use the slicers to filter by City Tier, Category, and Quarter
